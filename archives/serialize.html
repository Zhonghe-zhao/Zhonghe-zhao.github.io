json的序列化与反序列化

序列化：将 Go 中的结构体、切片、map 等数据结构转换为 JSON 格式的字符串。

作用：方便数据在网络传输、文件保存等情境中使用。

反序列化：将 JSON 格式的字符串转换为 Go 的数据结构。

作用：方便从 JSON 中提取数据并在程序中使用。

}
